1. fetch package
fetch

A dart/flutter package for making network requests.

Features

  • Get
  • Post
  • Put
  • Delete
  • Patch

Usage

To use the package add this to your pubspec.yaml

dependencies:
  fetch: ^0.0.1

Then import the package

import 'package:fetch/fetch.dart';

void main() async{
    // Use the package
}
